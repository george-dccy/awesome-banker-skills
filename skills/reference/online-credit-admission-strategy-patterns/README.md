# 线上授信准入策略建议

## 这个 skill 解决什么

它解决的不是“直接决定谁能贷”，而是：

- 让准入策略章节更有章法
- 让 Agent 知道应先看哪些维度
- 让输出停留在可参考建议，而不是越界成内部准入政策

## 当前优先场景

- 对公供应链场景
- 核心企业上下游场景
- 平台类交易场景
- 具备一定数据沉淀和交易闭环的线上化业务

## 不直接做什么

- 不直接给授信结论
- 不直接给阈值、分数线和最终准入条件
- 不替代内部政策和模型

## 公开依据方向

- 互联网贷款监管规则
- 助贷业务监管规则
- 个人贷款和流动资金贷款管理的通用要求
- 商业银行和大型银行对供应链金融、线上化场景金融的公开介绍
