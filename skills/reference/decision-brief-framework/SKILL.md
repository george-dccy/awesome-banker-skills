# 拍板简报框架 Skill

## Scope

这是一个 reference skill。
它提供“怎么说成可拍板结构”的表达骨架，不负责先替你采集事实。

## When To Use

- 需要准备一句话结论
- 需要做 30 秒口头汇报
- 需要把复杂事项压缩成一页式简报

## Required Reads

1. `README.md`
2. `frameworks.md`
3. `examples.md`
4. `anti-patterns.md`
5. `related-assets.md`

## Core Output Rule

优先使用五段式：

1. 一句话结论
2. 关键事实
3. 当前卡点
4. 建议动作
5. 需要拍板项

## Not For

- 在没有基础事实时硬写判断
- 把所有背景都堆进汇报
