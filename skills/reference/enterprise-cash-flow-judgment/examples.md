# Examples

## 示例 1：制造业客户 — 经营现金流持续改善

**现象（客户提供数据）：**

- 近三年净利润：1000万 / 1200万 / 1500万
- 近三年经营现金流：300万 / 800万 / 1400万
- 近三年货币资金：2000万 / 2500万 / 3000万
- 银行贷款余额：5000万（三年基本持平）

**框架判断：**

- 经营现金流逐年改善，与利润增长同步 → 利润质量好
- 货币资金持续增加 → 资金沉淀健康
- 债务不增加但经营现金改善 → 自给我偿债能力增强
- 经营现金流 / 总债务 ≈ 28%（1400/5000） → 利息覆盖有余

**结论：**
这是银行偏好的"现金牛"型客户，偿债能力稳定，可以进一步了解是否有新的资金需求。

## 示例 2：贸易类客户 — 经营现金流为负但有解释

**现象（客户提供数据）：**

- 近三年净利润：800万 / 900万 / 1000万
- 近三年经营现金流：-2000万 / -1500万 / -800万
- 近三年应收账款：5000万 / 7000万 / 8500万
- 营业收入增速：10% / 15% / 12%

**框架判断：**

- 经营现金流为负，但逐年改善 → 不是红色警戒
- 应收账款增速（40% / 21% / 18%）高于营收增速 → 回款质量下降
- 负现金流主要来自应收账款堆积 → 不是业务失血，是扩张中的账期管理问题
- 需要追问：下游核心客户是谁？账期结构是怎样的？

**结论：**
这不是"差客户"，是需要加强应收账款管理的客户。可以考虑用供应链金融产品帮助其加速回款。

## 示例 3：建筑类客户 — 融资现金流异常

**现象（客户提供数据）：**

- 近三年经营现金流：-500万 / -800万 / -1200万（持续为负）
- 近三年新增借款：3000万 / 4500万 / 6000万
- 近三年固定资产购建：200万 / 180万 / 150万

**框架判断：**

- 经营现金流持续为负 → 主营业务入不敷出
- 新增借款持续增加但主业未扩张 → 借款用于填补经营亏空
- 固定资产未增加 → 不是扩张型投资
- 融资现金流完全依赖借款 → 资金链高度依赖银行续贷

**结论：**
高度预警信号。建筑类客户本身有垫资特征，需要穿透看：项目回款节点是否明确？是否有明确的回款计划？当前续贷是否稳定？这类客户如果新增授信，必须要求项目回款作为第一还款来源。

## 示例 4：成熟期企业 — 三流平衡健康

**现象（客户提供数据）：**

- 经营现金流：5000万（正）
- 投资现金流：-2000万（购建固定资产，正常维护性投资）
- 融资现金流：-3000万（偿还到期银行贷款）
- 货币资金：8000万（稳定）

**框架判断：**

- 经营现金完全可以覆盖投资和融资 → 自给自足能力强
- 主动偿还债务 → 财务风格稳健
- 没有新增借款需求 → 短期内不是贷款目标客户
- 可以关注其是否有新的扩张计划

**结论：**
优质稳定客户，但不是当期的融资需求方。可以建立长期关系，关注其上下游供应链金融机会。
