# 企业现金流判断 Reference Skill

## 用途

从银行信审视角，对企业财报中的现金流信息做出专业判断。
判断重心不是"利润好不好"，而是"到期的贷款能不能用现金还"。

## 适用场景

- 客户拜访前的企业现金质量评估
- 信审报告中的现金流分析段落
- 客户融资需求与现金状态的匹配判断
- 向上汇报中的财务判断支撑

## 边界

- 这是现金流信号的判断框架，不是具体授信结论
- 不替代审批流程中的评分卡或评级模型
- 不承诺具体的贷款金额或定价
- 需要结合行业、周期、担保等综合判断

## 不负责什么

- 详细的财务预测模型
- 现金流折现计算
- 行业对比基准数据
- 最新监管政策的即时解读
