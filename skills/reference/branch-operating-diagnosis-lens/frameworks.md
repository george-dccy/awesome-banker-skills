# 判断框架

## 1. 先看结果，不先解释

经营分析的第一步不是解释原因，而是先判断：

1. 当前目标完成得怎么样
2. 当前主要结果指标是亮点还是短板
3. 这种变化是阶段波动，还是持续趋势

如果“结果判断”本身都没站稳，后面的归因通常会跑偏。

## 2. 经营分析的最小路径

推荐先按这四层看：

1. **结果**
   看目标、增量、同比、环比、绝对值
2. **结构**
   看客户结构、产品结构、区域结构、客群结构
3. **原因**
   看是过程动作、资源配置、资产质量还是外部环境因素
4. **动作**
   看哪些问题值得立刻推进，哪些问题要先补数再判断

简单说，就是：

`结果 -> 结构 -> 原因 -> 动作`

## 3. 先抓哪几类核心指标

公开经营材料里，银行常用来观察经营质量的指标通常包括：

- 资产负债规模和增速
- 存款和贷款增长及结构
- 净息差、净利息收入、非息收入
- 成本收入比
- 不良贷款率、拨备覆盖率等资产质量指标
- 普惠、小微、涉农等重点领域投放情况

对分支行和网点的轻量经营分析，优先可以抓：

- 存款
- 贷款
- 中收
- 利润或营收贡献
- AUM
- 客户数 / 活跃客户
- 新户 / 流失

## 4. 结果问题和结构问题怎么区分

### 更像结果问题

- 总量没达标
- 增速明显掉队
- 与目标或同业相比差距大

### 更像结构问题

- 总量还在，但高价值客户、重点产品、核心客群占比恶化
- 增长来自低质量或不可持续部分
- 客户和产品过度集中

### 更像过程问题

- 触达、拜访、跟进、协同、转化节奏弱
- 管理动作和资源投入跟不上

### 更像外部环境问题

- 区域产业波动
- 政策变化
- 行业风险暴露
- 同业竞争挤压

## 5. 亮点和短板不要只看绝对值

经营分析里最常见的误区，是只看“多了还是少了”。

至少还要一起看：

- 同比
- 环比
- 与目标比
- 与结构比
- 与质量比

比如：

- 贷款增长是亮点，但如果不良抬升、集中度升高，就不能只当亮点
- 存款增长不错，但如果主要来自高成本负债，也要单独标记结构压力

## 6. 先给初判，再列缺口

轻量经营分析最稳的写法不是“我已经判断完了”，而是：

- 当前初判：
- 支撑依据：
- 还缺哪些关键补数：
- 哪些结论暂不宜下：

## 7. 公开依据映射

- 2025 年一季度银行业保险业主要监管指标继续使用总资产、贷款投放、普惠小微、涉农、不良贷款率等指标观察行业整体运行
- 大型银行公开业绩说明中，通常同时看贷款和存款增长、净息差、成本收入比、非息收入占比、资产质量和结构优化
- 因此分支行经营分析的公开可复用框架，也应兼顾“结果、结构、质量、成本、风险、增长持续性”几个视角
