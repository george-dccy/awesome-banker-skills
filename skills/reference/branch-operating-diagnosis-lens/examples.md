# 示例

## 示例 1：先做结果判断

输入：

- 存款同比增长
- 贷款增量不错
- 中收承压
- 自评认为“整体还可以”

推荐输出方向：

- 先判断亮点和短板分别是什么
- 不要用“整体还可以”直接替代经营结论
- 先把“规模增长”和“收入承压”拆开看

## 示例 2：先拆结构，再讲原因

不好的写法：

- “贷款没上去，主要因为市场不好。”

更稳的写法：

- “当前贷款结果承压，但还需要继续拆客户结构、产品结构和过程动作，才能判断是外部环境主导，还是资源投入和推进节奏问题。”
