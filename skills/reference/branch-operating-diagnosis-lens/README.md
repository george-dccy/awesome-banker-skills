# 经营分析视角

## 这个 skill 解决什么

它帮助你先站稳一个经营分析视角，而不是一上来就报数字或堆结论。

重点是先回答：

- 当前结果到底算好还是不好
- 问题主要出在结果、结构、过程还是外部环境
- 下一步最该补哪几项数据

## 适用输入

- 阶段目标
- 存款、贷款、中收、利润、AUM、客户数、活跃度等关键数据
- 去年同期、上期或预算对比
- 自我评价或现场判断

## 输出边界

- 适合输出判断框架、初步亮点/短板、待补数项
- 不适合替代内部经营分析系统和正式经营考核

## 公开依据方向

- 金融监管总局公开监管指标
- 商业银行公开年报、半年报、业绩发布中的经营分析表述
- 大型银行在公开材料中常用的盈利能力、资产质量、负债成本、结构优化等分析口径
