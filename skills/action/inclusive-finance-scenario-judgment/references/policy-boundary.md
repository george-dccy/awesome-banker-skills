## 3. 监管红线（绝对禁止）

以下情况**绝对不可**判断为普惠金融场景：

- **无真实经营**：无营业执照、无实际业务、无经营场所的"空壳"申请
- **贷款资金流向不合规**：流入房市、股市、理财等禁止领域
- **虚假材料**：伪造经营数据、伪造上下游合同、伪造贸易背景
- **关联交易伪装**：通过虚构关联交易套取普惠贷款
- **禁入行业**：黄赌毒、洗钱、恐怖融资等违法行业
- **产能过剩行业**：国家明令淘汰的落后产能行业（需参考最新产业政策）

## 4. 常见合规边界

### 边界判断

| 场景 | 是否属于普惠 | 边界说明 |
|------|-------------|---------|
| 电商平台个人店铺（有实际经营） | 可能 | 需判断是否属于个体工商户或有营业执照的经营主体 |
| 网红/MCN机构 | 需具体判断 | 有正规营业执照的可能属于小微企业 |
| 跨境电商 | 需具体判断 | 有实际出口经营的可能属于普惠 |
| 加盟连锁店（独立法人） | 需具体判断 | 各加盟商是否为独立法人、独立核算 |
| 农户联合社 | 可能 | 需判断是否符合农业经营主体定义 |
| 农业合作社 | 可能 | 需判断是否符合小微企业或农业经营主体标准 |

### 合规核查要点

- 经营真实性：是否有实际业务场所、合同、发票、流水
- 行业合规性：是否属于国家支持行业，有无经营资质
- 贷款用途合规：资金不得流向禁止领域
- 借款人资质：是否为完全民事行为能力人、有无重大违约记录

## 5. 政策更新提示

- 国家产业政策调整时，重点支持行业可能变化
- 监管对普惠金融的考核口径可能调整（如两增两控的具体指标）
- 各地可能有差异化的区域政策

**来源说明**：本参考文件中的政策描述基于公开政策文件的通用框架。具体政策细节以银保监会、人民银行最新发布的文件为准。
