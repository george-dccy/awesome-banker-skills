# 财报基础知识包

## 用途

帮助银行对公客户经理、风险管理者等，在客户接触、信用分析、需求判断等场景中，建立对财务报表的专业理解。

三大报表是银行人与企业对话的基础语言。理解财报结构，才能问出正确的问题。

## 与 reference skill 的分层

- **本包**：三大报表的专业知识支撑层，回答「这是什么」「怎么看」
- **reference skill**（如 `enterprise-cash-flow-judgment`）：回答「用什么框架判断」「优先看什么」

本包不提供分析框架，只提供可信赖的事实基础。

- 不提供税务处理细节
