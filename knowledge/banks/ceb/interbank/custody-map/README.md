## 托管业务分类总览（7类）

来源：[光大银行公司业务页](https://www.cebbank.com/site/gsyw/index.html)

| 序号 | 托管类别 | 覆盖范围 |
|------|----------|----------|
| 1 | 证券投资基金托管 | 公募基金、私募基金 |
| 2 | 保险资产托管 | 保险资产管理计划 |
| 3 | 银行理财托管 | 理财产品资金托管 |
| 4 | 信托财产托管 | 单一信托、集合信托 |
| 5 | 年金基金托管 | 企业年金、职业年金 |
| 6 | 证券相关托管 | 券商客户资产、交易结算资金、QDII/QFII |
| 7 | 其他托管 | 资管行政外包 |

## 1. 证券投资基金托管

| 类型 | 说明 |
|------|------|
| 公募基金托管 | 面向公开募集的证券投资基金 |
| 私募基金托管 | 面向非公开募集的私募投资基金 |

**服务内容**：
- 资金保管
- 净值核算复核
- 投资监督
- 信息披露支持

## 2. 保险资产托管

| 类型 | 说明 |
|------|------|
| 保险资产管理计划托管 | 保险资产管理公司发行的资产管理计划 |

**服务内容**：
- 独立账户资金保管
- 投资资产验证
- 精算报告支持

## 3. 银行理财托管

| 类型 | 说明 |
|------|------|
| 理财产品资金托管 | 商业银行理财子公司发行的理财产品 |

**服务内容**：
- 理财资金独立托管
- 产品端到端资金监督
- 收益分配核算

## 4. 信托财产托管

| 类型 | 说明 |
|------|------|
| 单一信托托管 | 面向单一委托人的信托计划 |
| 集合信托托管 | 面向多个委托人的集合信托计划 |

**服务内容**：
- 信托财产独立性保障
- 信托资金保管与划付
- 受托人履约监督

## 5. 年金基金托管

| 类型 | 说明 |
|------|------|
| 企业年金及企业年金基金托管 | 企业及其职工建立的补充养老保险 |
| 职业年金及职业年金基金托管 | 机关事业单位及其工作人员的职业年金 |

**服务内容**：
- 年金基金资产保管
- 企业/职业年金计划资金监督
- 待遇支付核算

## 6. 证券相关托管

| 类型 | 说明 |
|------|------|
| 证券公司客户资产托管 | 证券公司客户委托的资产独立存管 |
| 客户交易结算资金存管 | 证券公司客户交易结算资金的银行存管 |
| QDII托管 | 合格境内机构投资者境外投资资产托管 |
| QFII托管 | 合格境外机构投资者境内投资资产托管 |

**服务内容**：
- 客户资产与券商自有资产分账管理
- 交易结算资金安全存管
- 跨境投资资产独立保管

## 7. 其他托管

| 类型 | 说明 |
|------|------|
| 资管行政外包 | 资产管理机构后台运营行政服务外包 |

**服务内容**：
- 净值核算、清算
- 注册登记（TA）支持
- 信息披露与报表
